# 🎓 GPA 변환 & 시간표 추천 Android 앱

## 🔹 기능
- 4.3 기준 학점 → 4.5 기준 변환 기능
- 입력된 시간표 기반 자동 추천 기능

## 🔹 기술 스택
- Android (Java), SQLite
